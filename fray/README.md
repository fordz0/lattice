# Fray

Distributed threads for Lattice. Fray is a local-first forum app that runs on your machine and syncs over the Lattice p2p network.

## Install

Make sure you have the Lattice daemon running, then:

`lattice install fray`

Then visit `fray.loom` in a Lattice-enabled browser.

## Run manually

`./fray`

## Access

Visit `fray.loom` in a browser with the Lattice extension installed.
