Setup required

Firefox needs to know .loom is a real address, not a search query and must trust Lattice's local HTTPS certificate authority.

  1. Click the button below to open about:config.
  2. Search for this preference:
browser.fixup.domainsuffixwhitelist.loom
  1. Click the + button on the right to add it (defaults to true).
  2. Download the local CA certificate while lattice-daemon is running.
  3. In Firefox: Settings → Privacy & Security → Certificates → View Certificates → Authorities → Import. Select lattice-local-ca.pem and trust it for websites.
  4. Come back here and click Done.
Make sure lattice-daemon is running before browsing.
