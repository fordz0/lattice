Setup required

Firefox needs to know .loom is a real address, not a search query and must trust Lattice's local HTTPS certificate authority.

  1. Paste this into the Firefox address bar:
about:config
  1. Search for this preference:
browser.fixup.domainsuffixwhitelist.loom
  1. Click the + button on the right to add it (defaults to true).
  2. Download the local CA certificate while lattice-daemon is running.
  3. Paste this into the Firefox address bar:
about:preferences#privacy
  1. Go to Certificates → View Certificates → Authorities → Import. Select lattice-local-ca.pem and trust it for websites.
  2. Come back here and click Done.
Make sure lattice-daemon is running before browsing. Need help? See the getting started guide.
